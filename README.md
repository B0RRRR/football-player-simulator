# Football Player Simulator

Полноценный 2D-симулятор футбольной карьеры с элементами RPG, написанный на C++ с использованием библиотеки SFML.

В этой игре вы создаете своего футболиста и проходите путь от резервиста до суперзвезды мирового уровня. Вы можете играть на разных позициях (Нападающий, Полузащитник, Защитник, Вратарь), тренироваться, участвовать в национальных чемпионатах, еврокубках (Лига Чемпионов, Лига Европы) и получать вызовы в национальную сборную. 

Матчи симулируются в интерактивном 2D-движке: вы наблюдаете за игрой в реальном времени, а в ключевые моменты вступаете в игру через динамичные мини-игры (удары по воротам, отборы, пасы, сейвы).

## Основные фичи
- **Карьерный Хаб:** Календарь матчей, расписание тренировок, статистика и управление карьерой.
- **Интерактивный 2D Движок Матча:** Наблюдение за игрой команды с видом сверху и прямое участие в ключевых эпизодах через мини-игры.
- **Позиции:** Геймплей и мини-игры уникальны для каждой роли (Форвард, Полузащитник, Защитник, Вратарь).
- **Масштабные турниры:** Национальные лиги, Лига Чемпионов, Лига Европы и матчи национальных сборных.
- **Умный тренер и Усталость:** Ваше место в стартовом составе зависит от формы, энергии и доверия тренера.
- **Система трансферов и контрактов:** Переходите в топовые клубы по мере роста рейтинга.

## Требования (Зависимости)

Для компиляции и запуска проекта необходима библиотека **SFML** (Simple and Fast Multimedia Library) версии 2.5 или выше.

**Установка на Ubuntu / Debian:**
```bash
sudo apt-get update
sudo apt-get install libsfml-dev
```

## Сборка и запуск

Проект использует систему сборки CMake, поэтому его легко скомпилировать на любой ОС (включая Windows, macOS и Linux).

1. Откройте терминал в папке проекта.
2. Создайте папку для сборки и перейдите в нее:
   ```bash
   mkdir build && cd build
   ```
3. Сгенерируйте файлы сборки с помощью CMake:
   ```bash
   cmake ..
   ```
4. Скомпилируйте проект:
   ```bash
   make -j4
   ```
5. Запустите игру:
   ```bash
   ./football_sim
   ```

## Структура проекта

Кодовая база проекта организована по профессиональным стандартам C++:
- `src/core/` — базовые классы приложения, настройки, менеджер ресурсов и `main.cpp`.
- `src/models/` — классы данных и бизнес-логики (Игрок, База данных, Менеджер Карьеры, Матч).
- `src/engine/` — логика интерактивной симуляции матча (`MatchEngine`).
- `src/screens/` — все экраны пользовательского интерфейса (Меню, Хаб, Матч, Тренировки, Статистика и т.д.).
- `src/ui/` — общие компоненты и темы UI.
- `assets/` — ресурсы (шрифты, изображения, звуки).
- `CMakeLists.txt` — конфигурация сборки проекта.
