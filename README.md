# Football Career Simulator

Простой 2D-симулятор футбольной карьеры, написанный на C++ с использованием библиотеки SFML.

В этой игре вы управляете футболистом. Матчи симулируются автоматически, но в ключевые моменты (когда мяч оказывается у вас) время останавливается, и вам нужно принять решение: пробить по воротам или отдать пас. Успех зависит от характеристик вашего игрока и выбранного уровня сложности.

## Требования (Зависимости)

Для компиляции и запуска проекта необходима библиотека **SFML** (Simple and Fast Multimedia Library) версии 2.5 или выше.

**Установка на Ubuntu / Debian:**
```bash
sudo apt-get update
sudo apt-get install libsfml-dev
```

## Сборка и запуск

Проект использует систему сборки CMake, поэтому его легко скомпилировать на любой ОС.

1. Откройте терминал в папке проекта.
2. Создайте папку для сборки и перейдите в нее:
   ```bash
   mkdir build && cd build
   ```
3. Сгенерируйте файлы сборки с помощью CMake:
   ```bash
   cmake ..
   ```
4. Скомпилируйте проект:
   ```bash
   make
   ```
5. Запустите игру:
   ```bash
   ./football_sim
   ```

## Структура проекта

- `src/` — исходный код игры (классы `GameManager`, `Match`, экраны и логика).
- `assets/` — ресурсы (шрифты, звуки).
- `CMakeLists.txt` — конфигурация сборки.
